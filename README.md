# My Notes App

A beautiful, fully functional note-taking web application built with modern web technologies. Create, edit, search, and organize your notes with an elegant and intuitive user interface.

## 🌐 Live Demo

**[View Live Demo](https://notes-app-woad-omega-46.vercel.app/)**

## 📋 About The Project

My Notes App is a responsive web application designed to help users keep their thoughts organized and accessible. Whether you're planning your day, brainstorming ideas, or managing important information, this app provides a seamless note-taking experience with a beautiful, modern interface.

### Key Features:
- ✨ **Create Notes** - Quickly create new notes with title and content
- 📝 **Edit Notes** - Modify existing notes anytime
- 🗑️ **Delete Notes** - Remove notes you no longer need
- 🔍 **Search Functionality** - Find notes instantly by title or content
- 🎨 **Color Coding** - Organize notes with 6 beautiful color options
- 💾 **Local Storage** - All notes are saved automatically to your browser
- 📱 **Fully Responsive** - Works seamlessly on desktop, tablet, and mobile devices
- ✨ **Beautiful UI** - Modern gradient design with smooth animations and transitions

## 🛠️ Tools & Technology

### Frontend Framework & Build Tools:
- **React 18** - JavaScript library for building user interfaces with component-based architecture
- **Vite 5** - Next-generation frontend build tool providing instant server start and lightning-fast HMR (Hot Module Replacement)
- **JavaScript (ES6+)** - Modern JavaScript for dynamic functionality

### Styling & UI:
- **TailwindCSS 3** - Utility-first CSS framework for rapid UI development with responsive design
- **PostCSS** - Tool for transforming CSS with plugins
- **Autoprefixer** - Automatically adds vendor prefixes to CSS rules for better cross-browser compatibility

### Storage:
- **LocalStorage API** - Browser's local storage for persistent data management without backend

### Deployment:
- **Vercel** - Cloud platform for deploying and hosting modern web applications

### Development Tools:
- **npm** - Package manager for managing project dependencies

## 💡 What This App Does

The **My Notes App** is a complete note-taking solution that allows users to:

1. **Add Notes** - Create new notes with a title and detailed content through an elegant modal interface
2. **Organize with Colors** - Choose from 6 different color themes to categorize and organize your notes visually
3. **Search & Filter** - Instantly search through all notes by title or content in real-time
4. **Edit Notes** - Click on any note to edit its title, content, or color
5. **Delete Notes** - Remove notes with a simple click on the delete button
6. **Persistent Storage** - All notes are automatically saved to your browser's local storage, so they persist even after closing the browser
7. **Responsive Design** - Access your notes from any device with full responsive layout support

## 🚀 Quick Start

### Installation

```bash
git clone <repository-url>
cd rect
npm install
```

### Development

```bash
npm run dev
```

The app will start on `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Build

```bash
npm run preview
```
